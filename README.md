## 📖 About This Repository

This is **NOT** a beginner tutorial on how to use `model.fit()` and `model.predict()` with high-level APIs.

This is a **full-stack AI/ML Engineering program** designed to teach you the internal mechanics, mathematical proofs, and system architectures of modern Artificial Intelligence. By the time you complete this curriculum, you will possess the ability to:
1. Read research papers and derive the underlying calculus and linear algebra.
2. Build algorithms like Neural Networks and Random Forests completely from scratch in pure Python.
3. Understand the memory management and C/C++ bindings of libraries like PyTorch, TensorFlow, and Scikit-Learn.
4. Architect multi-agent AI systems and deploy them with production-grade MLOps pipelines.

## 🧬 Module Structure

Inside every module folder, you will find a standardized structure designed for rigorous self-study:
- `README.md` - The core theory, mathematical derivations (in LaTeX), and Mermaid architecture diagrams.
- `code/` - Heavily commented Python scripts showing how the math becomes code.
- `assignments/` - Problems to solve (calculating gradients on paper, writing manual code).
- `solutions/` - Verified answers for all assignments.
- `datasets/` - Small toy datasets used for algorithms.
- `resources/` - Cheat sheets and PDF links.
